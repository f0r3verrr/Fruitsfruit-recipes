# Свежевыжатые соки нового поколения

Одностраничный сайт с рецептами соков-коктейлей в таре из фруктов.

## Технологии

- React 18
- Vite
- CSS3

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для production
npm run build
```

## Структура проекта

- `src/App.jsx` - главный компонент приложения
- `src/components/RecipeCard.jsx` - компонент карточки рецепта
- `src/data/recipes.js` - данные рецептов
- `src/App.css` - стили приложения
- `src/components/RecipeCard.css` - стили карточек рецептов
